### 1. 프로젝트 스캐폴딩 / 실행 환경

- **루트/구조**
  - 루트: `pgv3/`
  - 소스: `pgv3/src/`
  - 문서: `pgv3/docs/`
- **Poetry 설정**
  - `pyproject.toml`:
    - 패키지 이름: `container-management`
    - Python: `^3.12`
    - 의존성: `fastapi`, `uvicorn[standard]`, `pydantic`, `pydantic-settings`, `sqlalchemy`, `alembic`, `asyncpg`, `psycopg2-binary`, `aiomysql`, `pymysql`, `kubernetes`, `minio`, `pika`, `redis`, `httpx`, `python-jose[cryptography]`, `pyjwt`, `passlib[bcrypt]`, `aiofiles`, `python-dotenv`, `greenlet` 등
- **엔트리포인트**
  - `main.py`
    - `run_fastapi()`: FastAPI 단일 프로세스 실행
    - `run_all_processes()`: `ProcessManager`로 FastAPI/StateMachine/Monitoring 워커를 모두 실행

실행 예:

```bash
cd pgv3
poetry install
poetry run python main.py          # 단일 FastAPI
poetry run python -c "from main import run_all_processes; run_all_processes()"  # 멀티프로세스 (실험용)
```

### 2. 설정 / Config / application.yaml

- **`application.yaml`**

### 4. 공통 응답 / 예외 / 인증

- **응답 포맷**
  - `src/dto/response/api_response.py`
    - `ApiResponse[T]`: `result`, `errorCode`, `errorMessage`, `data`
  - `src/utils/common_response.py`
    - `success(data)`, `failure(error_code, message)`
    - `to_api_response(ApiResponse, status_code)`
    - `@common_response` 데코레이터: 엔드포인트 반환값을 `ApiResponse`로 감싸 `JSONResponse` 반환

- **예외 계층**
  - `src/exceptions/base.py`
    - `BaseCustomException(error_code, message, http_status)`
    - `AuthenticationException`, `AuthorizationException`
  - 도메인 예외:
    - `analysis_tool_exceptions.py` (`AnalysisToolException`, `AnalysisToolNotFound`)
    - `file_node_exceptions.py` (`FileNodeException`, `FileNodeNotFound`)
    - `acl_exceptions.py` (`AclException`, `AclNotFound`)
    - `app_access_exceptions.py` (`AppAccessException`, `InvalidAppCode`)

- **예외 핸들러**
  - `src/api/app.py`
    - `@app.exception_handler(BaseCustomException)`: `ApiResponse(result="0", errorCode, errorMessage, data=None)`
    - `@app.exception_handler(Exception)`: 예기치 못한 예외 → `errorCode = 클래스명`, 상태코드 500

- **인증/세션/JWT**
  - `src/security/session_manager.py`
    - `UserInfo(id, name, groups, is_admin)`
    - 인메모리 `SessionManager` (create/get/invalidate)
  - `src/security/jwt_utils.py`
    - `create_access_token(UserInfo)` → JWT (`sub`, `name`, `groups`, `is_admin`, `exp`)
    - `decode_access_token(token)` → `UserInfo`
  - `src/middleware/auth.py`
    - `get_current_user(Authorization: Bearer <token>)`
      - 먼저 JWT 디코딩 시도
      - 실패 시 세션 토큰으로 인식해 `SessionManager.get_user` 조회
      - 실패하면 `AuthenticationException`

### 6. 서비스 레이어 (주요 구현)

- **AppAccessLog**
  - DTO:
    - `AppAccessAddRequest(appCode, accessDate)`
    - `AppAccessSearchRequest(appCode/subId/userId/userName/filter/from/to/date/sort/offset/size)`
  - Enum:
    - `AppCode` (`DATA_IMPORTER`, `PGV2`, `ANLS_PROJECT`, `ANLS_TEMPLATE_*` 등)
  - 서비스: `src/services/app_access_service.py`
    - `add_access_log(req, user)`
      - `AppCode.find(req.appCode)`로 코드 유효성 검사
      - `AppAccessLog` 생성 (`app_code`, `sub_id`, `user_id`, `access_date`)
      - 세션 유저가 없으면 `user_id="UNKNOWN"`
    - `get_list_and_count(search)`
      - SQLAlchemy로 필터 조건/기간/정렬/페이징 구현
      - `filter` → `userId/userName/subId` LIKE 검색
      - `appCode` → comma-separated string → `AppCode` enum 리스트
      - 기간(`from/to/date`) → `access_date` 범위 필터
      - 정렬: `sort="key,ASC|DESC"` (`appCode/userId/userName/accessDate`)
      - 목록 + total 카운트 반환
    - `get_list(search)` → `items`만 반환

- **FileNode**
  - 엔티티: `FileNode` + `WasUser` 매핑
  - 서비스 스켈레톤: `FileNodeService`
    - `get_file_node_list_with_count`, `rename_file_node`, `get_exist_file_node_name`, `get_map_file_node_new_name_multi`, `update_file_object`, `create_file_node`, `create_file_node_multi` 등
    - Java `FileNodeService` / `FileNodeRepositoryCustomImpl` 로직 포팅을 위한 자리 확보

- **ACL**
  - `AclService` (스켈레톤): Meta ACL API 연동용 `MetaAclClient` 사용
  - ACL 조회/업데이트는 Meta 서버로 위임하는 구조

- **Analysis Tool**
  - `AnalysisToolService`/`BackupService` 스켈레톤:
    - 이름 중복, 생성/재신청/변경/만료/삭제/정지/재시작/백업 관련 메서드 구조 정의
    - 실제 컨테이너 생성/리소스 검증은 `ContainerClient`를 통해 수행하도록 설계

- **Common / IRIS / Time**
  - `CommonService`
    - `get_session_user(user: UserInfo)` → 세션 정보 dict
    - `get_config()` → `analysisTool` 설정 등 반환
    - `create_token(credentials)` → `create_access_token(UserInfo)`로 JWT 발급 (현재 dummy 사용자 기준)
  - `IrisService`
    - `status`, `get_route`, `event`, `heartbeat` 메서드 스켈레톤
  - `TimeService`
    - `get_server_time()`, `get_time_offset(...)`

### 8. 워커 / 이벤트 / ProcessManager

- **Event Queue**
  - `src/core/event_queue.py`
    - `EventType` (`CREATE/STOP/DELETE/BACKUP`)
    - `EventPayload(event_type, tool_id, user_id, data)`

  - `src/core/process_manager.py`

    - `start_fastapi()`, `start_statemachine_worker()`, `start_monitoring_worker()`
    - `start_all()`, `stop(name)`, `stop_all()`, `check_health()`

- **Workers**
  - `workers/fastapi_process.py`
    - 별도 프로세스로 FastAPI 앱 실행 (`uvicorn.run(create_app(), ...)`)
  - `workers/statemachine_worker.py`

  - `workers/monitoring_worker.py`
    - 주기적으로 상태 조회/DB 업데이트/이벤트 발행 (스켈레톤)

- **Actions**
